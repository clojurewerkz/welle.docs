Welle

An expressive Clojure client for Riak

Get Started! » Read doc guides » Join the Mailing List » Contribute »

Clojuric

Work with Clojure data structures. No unnecessary abstractions. Simple and easy.

Efficient

Welle has very low overhead thanks to the Riak Java client under the hood and heavy use of Clojure 1.3+ features (namely protocols)

Automatic Serialization

Just tell Welle content type of the data you store and it will automatically serialize and deserialize it (for popular formats: JSON, Clojure reader, gzipped JSON, UTF-8 text)

Riak 1.0+ Features

Besides fundamental K/V operations, Welle supports secondary indexes, map/reduce, link walking, both HTTP and Protocol Buffers transports

Batteries Included

Welle makes it easier to do map/reduce with JavaScript resources, provides clojure.core.cache protocol implementation, unit testing helpers and more

Built for the Future

Built from the ground up for Clojure 1.3+. Tested against 1.3, 1.4 and 1.5 master snapshots.
